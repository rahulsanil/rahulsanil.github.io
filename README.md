# Polished Static Resume Website

This package contains a professional single-page static resume website.

## Files

- `index.html` — open directly in a browser or upload to any static hosting service.

## Printing / PDF

Open `index.html` in Chrome or Edge and click **Print / Save as PDF**. The page includes A4 print styling.

## Notes

- No external libraries or internet connection required.
- Fully static HTML and CSS.
- Responsive for desktop, tablet and mobile.
